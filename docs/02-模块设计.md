# 模块设计

## 模块划分

---

### 1. 前端接口模块 (portal)

**职责**：实现与web页面、IM机器人的对接

**核心组件**：

#### web
**职责**：向web前端提供接口

---

### 2. 模型对接模块 (hub)
**职责**：收拢所有模型调用，统一管理不同模型的调用逻辑、响应处理

---

### 3. 配置管理模块 (config)

**职责**：统一管理应用配置

---

### 4. AGENT 模块 (agent)

**职责**：实现AI Agent的核心业务逻辑

**核心组件**：

#### memory
**职责**：实现记忆管理，包括短期记忆和长期记忆

---
