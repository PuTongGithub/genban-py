# 配置说明

## 配置文件概览

项目使用两种配置方式：
1. **JSON 配置文件** - 应用配置（模型列表、工具配置等）
2. **.env** - 环境变量（API Key 等敏感信息）

---

## 配置文件位置

### json配置文件

位于 `src/config/jsons/` 目录下：

```
src/config/jsons/
├── app_config.json   # 应用配置（模型、工具等）
└── prompts.json      # 提示词配置
```

### env环境变量文件

位于项目根目录：`.env`（需自行创建）

---

## 配置项说明

### app_config.json

#### models 模型配置

| 配置项        | 类型     | 说明         | 示例           |
| ---------- | ------ | ---------- | ------------ |
| key（键）     | string | 模型名        | "qwen-test"  |
| id         | string | 模型 ID      | "qwen3-32b"  |
| max_tokens | int    | 最大 token 数 | 131072       |
| api        | string | 调用的 API 服务 | "dash_scope" |
| default    | bool   | 是否为默认模型    | true         |

#### tools 工具配置

| 配置项                       | 类型     | 说明              | 示例                 |
| ------------------------- | ------ | --------------- | ------------------ |
| read_file_path_whitelist  | array  | 允许读取的文件路径白名单    | ["C:\\Users\\..."] |
| write_file_path_whitelist | array  | 允许写入的文件路径白名单    | ["C:\\Users\\..."] |
| shell_timeout             | int    | Shell 命令超时时间（秒） | 120                |
| zhipu_search_engine       | string | 智谱搜索引擎类型        | "search_std"       |

---

### prompts.json

| 配置项            | 类型     | 说明                 | 示例            |
| -------------- | ------ | ------------------ | ------------- |
| system_content | string | 系统提示词，定义 AI 的角色和行为 | "你是一个聊天助手..." |

---

## .env

### 环境变量列表

| 变量名               | 类型     | 必填  | 说明                    |
| ----------------- | ------ | --- | --------------------- |
| DASHSCOPE_API_KEY | string | 是   | 阿里云 DashScope API Key |
| ZHIPU_API_KEY     | string | 是   | 智谱 API Key（用于网络搜索）    |

### 示例

```
DASHSCOPE_API_KEY=sk-...
ZHIPU_API_KEY=sk-...
```
