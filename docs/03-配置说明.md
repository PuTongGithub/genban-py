# 配置说明

## 配置文件概览

项目使用两种配置方式：
1. **config.toml** - 应用配置（系统提示词、模型列表等）
2. **.env** - 环境变量（API Key 等敏感信息）

---

## config.toml

### 配置项说明

#### [hub] 配置组

| 配置项 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| system_content | string | 系统提示词，定义 AI 的角色和行为 | "你是一个聊天助手..." |
| model_list | array | 可用的模型名称列表 | ["qwen3-14b", "qwen3-32b"] |

## .env

### 文件位置

项目根目录：`.env`（需自行创建）

#### 环境变量列表

| 变量名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| DASHSCOPE_API_KEY | string | 是 | 阿里云 DashScope API Key |
