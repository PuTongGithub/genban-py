# 接入网络搜索工具 - 技术评估文档

## 需求背景

为 genban-py 项目接入联网搜索能力，使 AI 助手能够搜索互联网上的实时信息。

## 项目现状分析

### 架构概览

- **项目定位**：长期陪伴服务型个人 AI 助理
- **当前模型平台**：阿里百炼（DashScope）
- **工具实现模式**：继承 `Tool` 抽象类 → 在 `ToolCaller` 中注册

### 现有工具结构

```

├── tool_caller.py             # 工具调用管理器

	    ├── read_file_tool.py
	    ├── write_file_tool.py

```

### 现有 API 封装结构

```
hub/
├── api/
│   └── api_dash_scope.py      # 阿里百炼 API 封装
├── components/
```

## 智谱网络搜索 API 分析

### API 基本信息

- **接口地址**：`https://open.bigmodel.cn/api/paas/v4/web_search`
- **认证方式**：Bearer Token（API Key）
- **请求方式**：POST

### 核心参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| search_query | string | 是 | 搜索内容，限制70字符 |
| search_engine | string | 是 | 搜索引擎编码 |
| count | integer | 否 | 返回结果数量，1-50，默认10 |
| search_recency_filter | string | 否 | 时间过滤：oneDay/oneWeek/oneMonth/oneYear/noLimit |
| content_size | string | 否 | 内容长度：medium/high |

### 支持的搜索引擎

| 编码 | 说明 |
|------|------|
| search_std | 智谱基础版搜索引擎 |
| search_pro | 智谱高阶版搜索引擎 |
| search_pro_sogou | 搜狗 |
| search_pro_quark | 夸克搜索 |

### 返回结果结构

```json
{
  "id": "任务ID",
  "created": 1234567890,
  "request_id": "请求标识符",
  "search_result": [
    {
      "title": "标题",
      "content": "内容摘要",
      "link": "结果链接",
      "media": "网站名称",
      "icon": "网站图标",
      "refer": "角标序号",
      "publish_date": "发布时间"
    }
  ]
}
```

## 技术实现方案

### 步骤一：环境变量配置

**文件**：`.env`

**修改内容**：新增智谱 API Key

```
ZHIPU_API_KEY=your_api_key_here
```

---

### 步骤二：应用配置扩展

**文件**：`src/config/tomls/app_config.toml`

**新增配置项**：

```toml
[tools]
zhipu_search_engine = "search_std"      # 默认搜索引擎
```

---

### 步骤三：智谱 API 封装

**文件**：`src/hub/api/api_big_model.py`（新建）

**职责**：
- 封装智谱网络搜索 API 调用
- 从环境变量获取 API Key
- 处理 HTTP 请求和响应
- 未通过接口参数传入的 HTTP 请求参数均在文件内写死默认值

**接口设计**：

```python
def web_search(
    search_query: str,
    count: int,
    search_recency_filter: str
) -> dict
```

---

### 步骤四：实现 WebSearchTool

**文件**：`src/kernel/tools/tools/impl/web_search_tool.py`（新建）

**工具定义**：

| 属性 | 值 |
|------|------|
| name | web_search |
| description | 联网搜索工具，用于搜索互联网上的实时信息 |

**工具参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| search_query | string | 是 | 搜索关键词，限制70字符 |
| count | integer | 否 | 返回数量，默认配置值 |
| search_recency_filter | string | 否 | 时间过滤：oneDay/oneWeek/oneMonth/oneYear/noLimit |

**返回值**：智谱 API 返回的 `search_result` 对象列表

**错误处理**：搜索失败时返回包含错误信息的字符串，由 Agent 决定是否重试

---

### 步骤五：注册工具

**文件**：`src/kernel/tools/tool_caller.py`

**修改内容**：
- 导入 `WebSearchTool`
- 在 `__init__` 中调用 `self._registerTool(WebSearchTool())`

## 风险评估

### 潜在风险

1. **API 调用失败**：网络问题或 API 限流可能导致搜索失败
   - 解决方案：返回错误信息，由 Agent 决定是否重试

2. **Token 消耗**：搜索结果可能占用较多上下文 token
   - 解决方案：使用 `content_size=medium` 控制返回内容长度

3. **搜索质量**：不同搜索引擎返回结果质量可能不同
   - 解决方案：提供配置项，允许用户选择合适的搜索引擎
