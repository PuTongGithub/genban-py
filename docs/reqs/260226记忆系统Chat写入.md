# 记忆系统 Chat 写入开发方案

## 概述

本文档描述记忆系统第一阶段中，将对话流（Chat）写入文件系统的具体实现方案。

---

## 设计原则

1. **单一职责**：记忆模块只负责记忆的存储和读取，不关心业务逻辑
2. **IO 隔离**：所有文件/数据库 IO 操作封装在独立 storage 模块
3. **简单直接**：暂不考虑线程安全、缓存等复杂机制

---

## 架构设计

### 模块结构

```
src/
├── storage/                    # IO 操作模块
│   ├── __init__.py
│   └── file_storage.py         # 文件存储实现
├── kernel/
│   └── memory/
│       ├── __init__.py
│       └── memory_manager.py   # 记忆管理器
```

### 职责划分

| 模块 | 职责 |
|-----|------|
| `storage.FileStorage` | 底层文件 IO 操作：JSONL 读写、文件列表查询 |
| `memory.MemoryManager` | 业务层：Chat 序列化/反序列化、按时间组织文件 |

## 存储设计

### 存储路径

```
{data_dir}/memory/{user_id}/chat/
├── 20260226.jsonl
├── 20260227.jsonl
└── ...
```

- `data_dir`: 通过 `path_util.get_data_dir()` 获取
- `user_id`: 用户唯一标识，由调用方传入
- 文件名格式: `YYYYMMDD.jsonl`

### 文件格式

**JSON Lines 格式**，每行一个 Chat 对象：

```jsonl
{"source": "user", "id": "xxx", "time": "2026-02-26 14:30:15", "message": {...}}
{"source": "assistant", "id": "xxx", "time": "2026-02-26 14:30:20", "message": {...}}
```

class FileStorage:
    """文件存储底层操作类"""
    
    def append_to_jsonl(self, file_path: Path, records: List[dict]) -> None:

        读取 JSONL 文件所有记录

## 实现细节

### 1. Chat 序列化

使用 `dataclasses.asdict()` 将 Chat 对象转为字典：

```python
from dataclasses import asdict

chat_dict = asdict(chat)
json_line = json.dumps(chat_dict, ensure_ascii=False)
```

### 2. 日期提取

从时间戳中提取日期，确定存储文件：

```python
def _get_date_from_timestamp(timestamp: str) -> str:
    """从时间戳提取日期 (YYYYMMDD)"""
    # timestamp format: "2026-02-26 14:30:15"
    return timestamp[:10].replace("-", "")
```

### 3. 追加写入逻辑

```python
def append_chats(self, chats: List[Chat]) -> None:
    # 按日期分组
    date_groups = defaultdict(list)
    for chat in chats:
        date = self._get_date_from_timestamp(chat.time)
        date_groups[date].append(asdict(chat))
    
    # 分别写入各日期文件
    for date, records in date_groups.items():
        file_path = self._get_chat_file_path(date)
        self.file_storage.append_to_jsonl(file_path, records)
```

### 4. 时间范围查询逻辑

```python
def get_chats(self, start_time: str, end_time: str) -> List[Chat]:
    # 提取日期范围
    start_date = self._get_date_from_timestamp(start_time)
    end_date = self._get_date_from_timestamp(end_time)
    
    # 获取文件列表
    files = self.file_storage.list_jsonl_files(self.chat_dir, start_date, end_date)
    
    # 读取并过滤
    results = []
    for file_path in files:
        records = self.file_storage.read_jsonl(file_path)
        for record in records:
            if start_time <= record["time"] <= end_time:
                results.append(self._dict_to_chat(record))
    
    return results
```

## 错误处理

| 场景 | 处理方式 |
|-----|---------|
| 目录不存在 | 自动创建 |
| 文件不存在（读取）| 返回空列表 |
| JSON 解析错误 | 记录警告，跳过该条记录 |
| 文件写入失败 | 抛出异常 |

---

## 测试要点

1. **正常写入**：单条/多条 Chat 写入，验证文件内容
2. **跨日期写入**：验证是否正确分配到不同文件
3. **时间范围查询**：验证边界条件、跨文件查询
4. **异常场景**：空列表、无效时间格式、文件损坏
